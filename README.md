# coc-coverage

coc.nvim extension to show test coverage

## Install

`:CocInstall coc-coverage`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-coverage-keymap)`

## Lists

`:CocList demo_list`
